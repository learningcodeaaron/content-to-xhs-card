# AI Agent 护城河示例

这套示例基于已获作者授权的文章《AI Agent 公司到底有没有护城河？》，用于展示从长文章、卡片数据到可编辑 HTML 和 PNG 成品的完整结果。

- `source.md`：示例文章 Markdown。
- `card-data.json`：10 张卡片的 v2 结构化数据。
- `index.html`：无需后端、可离线打开的编辑器。
- `cards/`：10 张 1080×1440 PNG 成品。
- `demo.webm`：无配音操作演示。

在线体验：[nextwaylab.com/tools/xhs-card-demo/index.html](https://nextwaylab.com/tools/xhs-card-demo/index.html)

本地校验：

```bash
npm run test:example
npm run test:browser
```

示例内容可以用于评估 Skill 的结果，不代表任何投资或商业建议。
