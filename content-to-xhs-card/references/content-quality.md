# 内容解构与卡片质量规范

仅在分析文章、修订拆分方案或检查逐卡文案时读取。这是本 skill 最核心的能力规范:**读者不看原文,只翻卡片,也能完整读懂文章在讲什么、作者判断是什么、为什么成立、对自己有什么用。**

## 第一步:建立文章地图(动笔前必做)

在写任何一张卡之前,先用一段话回答四件事,写进确认稿顶部:

1. **核心问题**:文章要回答的那个问题(通常就是标题)。
2. **核心主张**:作者的最终结论(一句话,能独立成立)。
3. **原文模块**:文章由哪几个论证段落组成,各自承担什么作用。
4. **读者理解路径**:读者从第一张翻到最后一张,认知如何一步步递进。

每个重要原文模块必须映射到一张或多张卡;若省略某模块,要说明为什么(如"与另一模块重复""属于展开举例,可并入")。

## 第二步:给每张卡想清楚"三个一"

拆卡不是把段落切碎,而是**为每张卡设计一个能独立成立的表达单元**。每张卡必须同时满足:

- **一个主张**:标题就是本卡结论或冲突,不能是"背景""价值""其一"这种空壳。读者只看标题应能猜到本卡要说什么。
- **一组支撑**:2–4 条要点或一个对照/流程/分类关系,用来解释"为什么这个主张成立"。每条要点写成"判断｜解释"或"概念｜含义",不能是孤立名词。
- **一句记忆点**:底部金句推进结论、给出可复述的一句话,**不得复读标题**。

### 自解释硬规则(最重要)

想象读者手机相册里只存了这一张卡,没有上下文。逐条检查:

- 卡上出现的每个名词/简称,在本卡内是否能被理解?(如"独家动态数据"要在同卡给出解释或例子,不能只甩词。)
- 去掉标题后,正文能否让人反推出标题结论?能,才算解释充分。
- 本卡的要点是不是都在服务标题?跑题的要点删掉或移到别的卡。

## 第三步:先定关系,再选布局(禁止按"缺样式"选)

布局是内容关系的表达,不是装饰。按下面的决策顺序选,详见 `references/layout-catalog.md`:

| 原文里的关系 | 选用布局 |
| --- | --- |
| 单一判断 / 定义 / 一句话结论 | `statement.poster` |
| 同维度两方对照(A vs B、旧 vs 新、危险 vs 安全) | `comparison.split` |
| 有先后顺序的 3–6 步 / 一条推理链 | `process.timeline` |
| 持续反馈、越用越强的闭环 | `process.cycle` |
| 3–6 个并列要点(无顺序、无对照) | `list.cards` |
| 一个核心 + 若干支撑模块 | `framework.bento` |
| 两个维度交叉出的象限分类 | `framework.matrix` |
| 同单位、可比较的真实数值 | `data.ranking` / `data.dashboard` |
| 开篇钩子 | `cover.editorial` |
| 收束 + 关注 | `tail.cta` |

**选布局时问自己:这张卡的内容是"并列""对照""流程"还是"单一判断"?** 关系错配是最常见的翻车点——把对照关系塞进 list.cards、把单一判断硬凑成流程,都会让读者困惑。

## 第四步:严禁的做法(过去踩过的坑)

1. **禁止无意义装饰图形**:不得为了"填满画面"放与内容无关的圆圈、图形、占位词。`statement.poster` / `cover.editorial` 就是干净的大字版式,不需要中心图。如果一个图形里的文字换成任何别的词都不影响理解,它就是噪音,删掉。
2. **禁止跨卡重复同一视觉/词组**:多张卡出现同一组词(如"输入/理解/行动")或同一张图,说明内容没拆开或在硬凑图形。每张卡的主视觉必须承载本卡独有的信息。
3. **禁止关键词堆砌**:要点必须是完整判断,不能是"品牌;数据;入口"这种断词。
4. **禁止伪造数据关系**:没有真实同单位数值,不用排名/柱状/环图;不用格子面积暗示不存在的规模。
5. **禁止相邻卡表达同一结论**:总览卡只给地图,拆解卡必须补充新的解释或证据。

## 第五步:卡组连贯性

逐页检查:上一张提出了什么 → 本张解决什么 → 下一张为什么自然出现。6 张以上卡组通常至少覆盖 3 个布局族;相邻卡只有表达相同关系时才重复布局。

## 发布文案

- 发布标题(3–5 个)与正文进入 `deck.publication`,与逐卡文案一起确认。
- 发布正文必须直接总结原文核心观点,不介绍卡片、不堆平台套话;写法与禁用表达见 `references/publication-copy.md`。

## 交付前自检(逐条打勾,不过不交付)

1. 原文每个主要模块是否都被覆盖或已说明省略理由。
2. 每张卡是否满足"三个一"(一个主张 / 一组支撑 / 一句记忆点)。
3. **拿任意一张卡单独给一个没读过原文的人,他能否读懂这张卡在说什么。**
4. 是否存在无意义装饰图形或跨卡重复的视觉/词组(有则删改)。
5. 每张卡的布局是否匹配其内容关系(单判断/对照/流程/并列/分类)。
6. 删除任意一张中间卡,论证是否出现明显断层;若毫无影响,该卡多余。
7. 只翻卡片(不看原文、不看发布正文),能否回答:文章在讨论什么、核心判断是什么、为什么成立、对我有什么意义。
